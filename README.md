# weatherjs
> A simple weather app, utilising OpenWeatherMap API

## Requirements
* __Node__
* __Vite__

## Setup & Usage
Clone repo
```bash
$ cd github-repos/
github-repos$ git clone git@github.com/biobot-01/weatherjs
```
Install dependencies
```bash
$ cd weatherjs/
weatherjs$ npm install
```
Create .env file
```bash
weatherjs$ touch .env > "VITE_WEATHER_API_KEY={apiKey}"
```
Run the App
```bash
weatherjs$ npm start
```
